# fsquads <img src="man/figures/logo.png" align="right" height="138.5"/>

## Introduction

This is an R package for retrieving data from
<http://www.footballsquads.co.uk>.

### Installation

`devtools::install_github('tonyelhabr/fsquads')`.

## Quick Reference

    #> get_league_teams_meta
    #> get_leagues_meta
    #> get_team_players
